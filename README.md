### Project name.
  - The project name is Paddy Shop

### Short description of the project.
  - A paddy shop is a small-scale retail business that collects and sells paddy (unhusked rice) to local customers, wholesalers, and rice mills. The project     focuses on sourcing quality paddy from farmers, maintaining proper storage conditions, and ensuring smooth distribution. It aims to support local agriculture,provide fair prices, and create a reliable supply chain for rice production.

### 5 key features of the project.
  - 1. Quality Paddy Sourcing.
  - 2. Efficient Storage System.
  - 3. Fair Pricing & Transparency.
  - 4. Reliable Supply Chain.
  - 5. Community & Farmer Support.
